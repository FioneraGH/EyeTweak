# iTweak

Multi Tweaks on a jailbroken iDevice.

## Compatibility

All devices running iOS 13.2-13.3.1

### Build

Install [Theos](https://github.com/theos/theos).

Run `make package` and it will compile the deb in ./packages/ directory.
